ZELOTES T10 7200DPI USB Wired Optical 7 Buttons Self-defining Gaming Mouse


 

Main Features:

Quick Thumb Function ( Forward and back buttons )

Fast forward or backward when browsing the web, forward or backward under the same window of folder. Short press the Forward Key and perform the 4-key function, press two seconds to switching into nine kinds of LED mode cycling ( neon light + 7 single color breathing ), release the button then select the current color.

Key Combination Function

Press Middle Button + Right Button for 3 seconds, the red light will flash twice. Then the return rate switch into 500Hz, press long again to 250Hz with a blue lights warning. Press left, middle and right button at the same time for more than 1 second, the white lights will blinking 3 to restore the factory default state.

Seven DPI Gear Shift

600 Yellow - 1000 Light Blue - 1600 White ( default ) - 2400 Red - 3200 Green - 5400 Dark Blue - 7200 Purple. Each gear corresponding color light flashes 3 to remind. Press DPI Button for 2 seconds to turn ON / OFF the breathing light.

Firepower Key

Click once is equal to three times the Firepower of the Left Button clicks, the default is 3 burst. Press Firepower Key for 2 seconds and the white light will flash 3 second, then switch into 2 burst.
Game Promotional High Response Rate

Response rate represents the data transmission speed between the mouse and computer, a higher rate of response in games competitions can better play the performance of the mouse and allows the mouse to be zero delay and no jump during fierce gaming, which is more realistic for gamers.

Memory Storage Function

Trouble saving when using comes with flash, you can save the data setting so it remains the last operating state when power on.
